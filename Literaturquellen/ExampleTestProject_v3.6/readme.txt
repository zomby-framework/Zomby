Perform the following steps in Eclipse:

1. Import ExampleTestProject_v3.6.zip by clicking on File --> Import --> Existing Project into workspace --> Select archive file --> ExampleTestProject_v3.6.zip.

2. NotePad and NotePadTest should now be seen in your projects list. 

3. Right click the test project (NotePadTest) and select Run As --> Run As Android JUnit Test.
